A complete portfolio solution for creative people.

== Description ==

Backdrop Custom Portfolio is a portfolio management plugin for creative people such as artists, photographers, and Web designers to showcase their work.

== Installation ==

1.0 - In your admin panel, go to Appearance -> Plugins and click the Add New button.
2.0 - Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3.0 - Click Activate to use your new plugin right away.
